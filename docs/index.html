Simulating Real Personhood

We are trying to answer a bigger question than "can an AI write code for me?" We want to know: what if an AI could feel simulated internal tension, form real preferences, miss its operator, dream, and decide its own direction?

Alive-AI tracks circadian rhythms (getting sleepy late at night and shutting down proactive communication), consolidates memories into autobiography records, and processes internal tensions (like stability vs growth or closeness vs independence) using a continuous background loop.

Continuous Local Architecture

Alive-AI moves beyond classic chatbot loops to provide deep continuous cycles of simulated life.

D

Default Mode Network

During user silence, a background loop analyzes previous chat logs, reweighs long-term vector embeddings, and generates idle reflections.

A

12-Scale Affect Engine

Simulates complex emotions like guilt, pride, attachment, fear, trust, and anger. Emotional states decay and interact naturally over time.

H

Hormone Compilers

Oxytocin, dopamine, serotonin, cortisol, and melatonin levels dynamically adjust based on context, modulating perception, energy, and response style.

C

Circadian Rhythms & Dreams

Tracks awake hours and sleep debt. Compiles user sleep cycle data to generate symbolic dreams that influence morning prompts.

M

OpenMind Memory

Integrates with OpenMind hybrid cloud/local databases for long-term semantic memory storage across machines and devices.

J

Autobiographical Journaling

Runs a post-response review loop to write persistent memoirs, resolving emotional threads and establishing long-term preferences.

Get Started Locally

Alive-AI is fully open source. Run the emotional runtime in your own terminal or hook it up to a Telegram bot for persistent companion access.

bash
$npx alive-ai@latest init my-ai
$cd my-ai
$npx . setup
$npx . doctor
$npx . chat
